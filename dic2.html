About Dictionary

Published, Oct 31, 2019.

  • A Dictionary is a generic class of C# that is used to represent the collection of (key, value) pairs.

  • Dictionary class is present in System.Collections.Generic namespace

  • When creating a Dictionary , we need to specify the type for key and values.

  • Keys in the dictionary must be unique and cannot be null.

  • Dictionary provides fast lookups for values using keys.

  • Dictionary has three properties- Count, Key and Values.

Need of Dictionary

  • It is faster than HashTable.

  • It will generate error if we try to use a key that is not exist in the collection.

  • It is a generic type so we can use any type of data.

In TryGetValue() function we will pass two parameters. One is In Parameter and other is out Parameter.

The In Parameter is the key we want to search. The out parameter is the value associated with the key.

The return type is Boolean. If the key is found, It will return True and the Value is passed through out parameter or else it will return false.
